# Tech Meditations

Personal blog at [ealejo.dev](https://ealejo.dev). Writing about GenAI, cloud architecture, and the evolving tech landscape.

Built with [Astro](https://astro.build) and Tailwind CSS v4. Deployed on Cloudflare Pages.

## Commands

| Command | Action |
|---|---|
| `npm install` | Install dependencies |
| `npm run dev` | Start dev server at `localhost:4321` |
| `npm run build` | Build for production to `./dist/` |
| `npm run preview` | Preview production build locally |
